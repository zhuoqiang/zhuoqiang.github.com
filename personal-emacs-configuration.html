Personal Emacs Configuration

Emacs is known to be one of the most powerful editor in the world. Yet its power is hidden in the absence of good configuration.

Emacs configuration is hard and fun. After quite a while hard time fighting against elisp I finally have some fun with my Emacs and I just open source my personal Emacs configuration hoping it's helpful to you.

The functions in the configuration are minimal for the sake of simplicity:

  • Chinese & English font setting with font scale function
  • key re-mapping on Mac
  • programming language mode setting
  • Jump functions
  • some emacs themes
  • auto restore files when re-open
  • save typing using snippet and auto insert
  • maxmize frame when startup emacs

It works on Emacs 24 under Mac and Windows (not tested under Linux though). All you need to do is checkout the code to ~/.emacs.d/. If you already have .emacs file under home directory, you have to remove it so that Emacs could use ~/.emacs.d/init.el instead.

It could be a good start to enhance your emacs further, good luck hacking!
